# Raiden
Raiden is a scrolling shooter arcade game wich was developed in September 1990 by Seibu Kaihatsu.
### CONTROLS

**W:** MOVE UPWARDS

**A:** MOVE LEFT

**S:** MOVE DOWNWARDS

**D:** MOVE RIGHT

**SPACE:** SHOOT

**TAB:** NEXT STAGE

**F1:** DEBUG MODE

**1:** 1 PLAYER MODE

**2:** 2 PLAYERS MODE
